Portfolio Edufy

Project Overview

Edufy is an advanced E-learning platform aimed at providing a comprehensive and engaging learning experience for students and instructors. The project leverages the latest web technologies and AI integrations to create a dynamic educational environment. With features like custom content management, responsive user interfaces, and AI-powered assistance, Edufy stands out as a robust solution for modern education.

Project Details

To build a scalable E-learning platform that facilitates seamless communication, content delivery, and interactive learning experiences.

  • Frontend: HTML, CSS, Bootstrap, JavaScript, React
  • Backend: PHP, Laravel
  • Database: MYSQL
  • API integration: OpenAI (ChatGPT)
